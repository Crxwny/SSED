# a_9_1 BruteForce (DVWA Unterricht)

Dieses Beispiel testet Username/Password-Kombinationen gegen `login.php` einer DVWA-Instanz.

## Dateien

- `bruteforce.js` - Script mit `fetch` (GET + POST)
- `usernames.txt` - 10 Usernamen
- `passwords.txt` - 10 Passwoerter

## Starten

```bash
node bruteforce.js
```

Im Unterrichts-Setup sollte `admin` / `password` gefunden werden.

## Hinweis

Nur in deiner eigenen Testumgebung verwenden (z. B. DVWA im Labor).
